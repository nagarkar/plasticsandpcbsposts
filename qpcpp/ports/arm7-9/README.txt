About The QP-ARM7-9 Port
========================
This directory contains the QP/C ports to the "classic" ARM7/ARM9 processors.
The following ports are provided:

arm7-9
|
+-qk      - port to the preemptive QK kernel
| +-gnu   - port with the GNU-ARM toolset
| +-iar   - port with the IAR-ARM toolset (IAR EWARM)
|
+-vanilla - port to the cooperative Vanilla kernel
| +-gnu   - port with the GNU-ARM toolset
| +-iar   - port with the IAR-ARM toolset (IAR EWARM)
